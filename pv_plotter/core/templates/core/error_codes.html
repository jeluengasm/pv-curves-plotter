Códigos de error

  1. Inicio
  2. FAQ

A continuación, se enunciarán los codigos de error que se notifican en el panel de notificaciones de la aplicación y su solución correspondiente.


  • Error 001: No tiene permisos de acceso en el puerto

    Microsoft Windows:


    • Verifique que el trazador esté conectado correctamente.
    • Verifique que el puerto serial (COM) corresponde con el visualizado en el Administrador de dispositivos.
    • Verifique si el perfil de usuario de la sesión actual posea permisos sobre el uso de puertos seriales.

    Linux:


    • Verifique que el trazador esté conectado correctamente.
    • Verifique que el puerto serial (TTYACM) corresponde con el identificado con el comando dmesg en el terminal.
    • Verifique si el perfil de usuario de la sesión actual posea permisos sobre el uso de puertos seriales.Conceda permisos de uso al puerto serial identificado, por medio del comando sudo chmod 777 /dev/ttyACM<X>.
  • Error 002: El puerto <...> no responde

    Microsoft Windows:


    • Verifique que el trazador esté conectado correctamente.
    • Verifique que el cable USB conectado sea de transmisión de datos y no de alimentación únicamente.
    • Verifique que el puerto serial (COM) corresponde con el visualizado en el Administrador de dispositivos.
    • Verifique si el perfil de usuario de la sesión actual posea permisos sobre el uso de puertos seriales.

    Linux:


    • Verifique que el trazador esté conectado correctamente.
    • Verifique que el cable USB conectado sea de transmisión de datos y no de alimentación únicamente.
    • Verifique que el puerto serial (TTYACM<X>) corresponde con el identificado con el comando dmesg en el terminal.
    • Conceda permisos de uso al puerto serial identificado, por medio del comando sudo chmod 777 /dev/ttyACM<X>.
  • Error 003: Archivo no válido

    • Verifique que el archivo a importar sea consistente con la estructura de archivos CSV recomendada en la sección "Aplicación/Archivos CSV".
    • Todas las filas correspondientes a las columnas "Corriente", "Tensión", "Potencia" y "Temperatura" deben ser datos numéricos con el mismo separador decimal (coma o punto).
    • En caso de ser un archivo CSV con separador decimal coma (,), cada columna debe separarse con un punto y coma (;).
    • En caso de ser un archivo CSV con separador decimal punto (.), cada columna debe separarse con una coma (,).
  • Error 004: La carpeta de exportación automática no existe

    La dirección que colocó para exportar automáticamente no existe. Pruebe lo siguiente:


    • Haga clic en el botón [...]. Se desplegará una ventana para selección de una nueva ruta de destino.
    • Seleccione una ruta que exista en su equipo. Si lo considera necesario, puede crear una carpeta para almacenar las mediciones automáticas.
    • Guarde sus cambios haciendo clic en el botón "Guardar configuración".
