Descargar software

Para hacer un uso correcto de su trazador de curvas es necesario descargar el archivo ejecutable. A continuación se presentarán los pasos correspondientes:

Paso 1

Ingrese al siguiente link y abra a la carpeta "Aplicación": Link

Paso 2

Seleccione la carpeta "Dataset" (1) y la carpeta del sistema operativo donde realizará las pruebas correspondientes (2), luego haga clic en "Descargar" (3).

Paso 3

Una vez descargados los archivos, haga clic sobre el archivo comprimido (1) y descomprima los archivos (2) en la carpeta de su preferencia.

NOTA: NO guarde los archivos en una carpeta protegida por el sistema operativo (Por ejemplo, archivos de programa), de lo contrario el trazador no funcionará.

Paso 4

Una vez descomprimidos, le deben aparecer las carpetas "Dataset" y la del sistema operativo que eligió.

Última versión de la aplicación: 1.8.

Hardware

  • Trazador de curvas IVT2203.
  • Cable USB para transmisión/recepción de datos.
  • Panel solar (máx. Voc de 22 V, Isc de 2.7 A).
  • Conectores banana macho (rojo y negro).
  • Sensor de temperatura con conector de chasis 4 polos.
  • Computadora con puertos USB activos (2.0 en adelante).
  • Mínimo 2 GB de memoria RAM.
  • Resolución de pantalla mínima: 1366 x 768.

Software

Sistemas operativos compatibles:

  • Microsoft Windows 7/8/8.1 de 64 bits.*
  • Microsoft Windows 10 de 64 bits.
  • Linux (basado en Debian) de 64 bits.

* Para el correcto funcionamiento del trazador de curvas, es necesario instalar un driver. Haga clic aquí para conocer el procedimiento.
