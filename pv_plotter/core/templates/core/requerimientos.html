Hardware

  • Trazador de curvas IVT2203.
  • Cable USB para transmisión/recepción de datos.
  • Panel solar (máx. Voc de 22 V, Isc de 2.7 A).
  • Conectores banana macho (rojo y negro).
  • Sensor de temperatura con conector de chasis 4 polos.
  • Computadora con puertos USB activos (2.0 en adelante).
  • Mínimo 2 GB de memoria RAM.
  • Resolución de pantalla mínima: 1366 x 768.

Software

Sistemas operativos compatibles:

  • Microsoft Windows 7/8/8.1 de 64 bits.*
  • Microsoft Windows 10 de 64 bits.
  • Linux (basado en Debian) de 64 bits.

* Para el correcto funcionamiento del trazador de curvas, es necesario instalar un driver. Haga clic aquí para conocer el procedimiento.
